# 🎨 Syntax Roulette - AI文本生图工具

> 用文字描述你的想象，让AI为你创作图像

---

## ⚡ 快速开始（3秒启动）

### Windows用户（推荐）

双击运行 `启动应用.bat` 文件，应用会自动：
1. ✅ 检查Python环境
2. ✅ 安装必要依赖
3. ✅ 启动Web界面
4. ✅ 打开浏览器

### 手动启动

```bash
# 1. 安装依赖（仅首次需要）
pip install gradio pillow numpy

# 2. 启动应用
python main.py
```

启动后在浏览器访问：`http://localhost:7860`

| 功能 | 说明 |
|------|------|

## 💡 提示词技巧

### ✅ 好的提示词

**具体明确：**
```
一只橘色小猫坐在木窗台上，温暖的阳光透过玻璃照射，
毛发清晰，背景是模糊的绿色花园
```

**添加质量词：**
```
专业摄影, 高清, 8k, 细节丰富, 最佳质量, 电影般的光影
```

### ❌ 避免的写法
- ❌ 太简单：`一只猫`
- ❌ 太复杂：超过200字的长段落
- ❌ 矛盾：`白天的夜晚景色`

## ❓ 常见问题

### Q: 为什么生成的是彩色渐变图？
**A:** 当前为演示模式。要获得真实AI图像：
- 需要安装并连接ComfyUI服务
- 或替换为其他AI图像生成API

### Q: 如何提高图像质量？
**A:** 
1. 选择"高"或"超高"质量
2. 使用更大的图像尺寸
3. 提供详细具体的描述
4. 添加"高清"、"细节丰富"等关键词

### Q: 如何保存生成的图像？
**A:** 
- 点击"📥 下载图像"按钮
- 或右键点击图像选择"另存为"

### Q: 应用无法启动怎么办？
**A:**
1. 确认Python已安装（3.8+）
2. 重新安装依赖：`pip install gradio pillow numpy`
3. 查看终端错误信息
