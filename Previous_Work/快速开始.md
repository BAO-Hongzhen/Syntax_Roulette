# 🚀 快速开始指南

## ⚡ 最快启动（Windows）

**双击** `启动应用.bat` 文件，完成！

应用会自动：
1. ✅ 检查Python环境
2. ✅ 安装依赖（首次运行）
3. ✅ 启动Web服务
4. ✅ 打开浏览器

---

## 📋 手动启动

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动应用

```bash
python main.py
```

### 3. 打开浏览器

访问: `http://localhost:7860`

---

## 🎮 使用流程

### 步骤1: 生成句子

1. 选择句式模板
   - **简单句式**: 主语 + 动词 + 宾语
   - **详细句式**: 完整的修饰结构

2. 点击 **"🎲 随机生成句子"**

3. 查看生成结果
   - 完整句子
   - 句子结构分析

### 步骤2: 生成GIF动图

1. **调整参数**（可选）
   - 宽度/高度: 512x512（推荐）
   - 帧数: 16帧
   - 帧率: 8 FPS

2. **选择模式**
   - 演示模式: 快速预览（默认）
   - ComfyUI模式: 真实AI生成

3. **点击** "🎬 生成GIF动图"

4. **等待生成**
   - 演示模式: 1-2秒
   - ComfyUI模式: 30-60秒

5. **查看结果**
   - GIF自动显示
   - 可以下载保存

---

## 🎨 两种模式对比

### 演示模式

✅ **优点:**
- 无需额外配置
- 快速生成
- 测试功能

⚠️ **限制:**
- 彩色动画预览
- 非真实AI生成

**适合**: 测试、演示、快速体验

---

### ComfyUI模式

✅ **优点:**
- 真实AI生成
- 高质量动图
- 创意无限

⚠️ **需要:**
- 安装ComfyUI
- 本地运行服务
- 较长生成时间

**适合**: 正式创作、高质量输出

---

## 🔧 启动ComfyUI

### 安装ComfyUI

```bash
# 克隆仓库
git clone https://github.com/comfyanonymous/ComfyUI
cd ComfyUI

# 安装依赖
pip install -r requirements.txt

# 下载模型（可选，使用默认模型）
```

### 启动ComfyUI服务

```bash
# 在ComfyUI目录下运行
python main.py
```

服务启动后会显示:
```
To see the GUI go to: http://127.0.0.1:8188
```

### 在应用中启用

1. 确认ComfyUI正在运行
2. 在界面中勾选 **"使用ComfyUI生成"**
3. 开始生成

## 💡 使用技巧

### 生成更好的句子

1. **尝试不同句式**

   - 详细句式: 适合丰富场景

   ```

   ```

3. **批量生成**
   - 使用相同句子

   - 在历史记录中对比

## ❓ 常见问题

### Q: 应用启动失败？

**A**: 检查以下几点:
```bash
# 1. 检查Python版本
python --version  # 需要 3.8+

# 2. 重新安装依赖
pip install -r requirements.txt --upgrade

# 3. 查看错误信息
python main.py
```

---

### Q: ComfyUI连接失败？

**A**: 
1. 确认ComfyUI正在运行
   - 访问 http://127.0.0.1:8188
   - 应该能看到ComfyUI界面

2. 检查端口是否被占用
   - 默认端口: 8188
   - 可在ComfyUI启动时修改

3. 查看ComfyUI日志
   - 终端窗口的错误信息

---

### Q: 生成GIF失败？

**A**: 
1. **演示模式失败**:
   - 检查output目录权限
   - 重启应用

2. **ComfyUI模式失败**:
   - 检查ComfyUI运行状态
   - 查看终端错误日志
   - 尝试演示模式测试

---

### Q: 如何自定义词库？

**A**: 
1. **通过界面**: 查看"词库管理"部分

2. **通过代码**:
   ```python
   from modules.word_bank import WordBank
   
   word_bank = WordBank()
   word_bank.add_word("subjects", "your word")
   word_bank.save_to_file("data/custom.json")
   ```

3. **通过JSON文件**: 参考README的自定义词库部分

---

### Q: GIF保存在哪里？

**A**: 
- **演示模式**: `output/demo_*.gif`
- **ComfyUI模式**: `output/syntax_roulette_*.gif`

---

### Q: 如何提高生成速度？

**A**: 
1. 减少帧数（8-12帧）
2. 降低分辨率（256x256或384x384）
3. 使用演示模式快速测试
4. ComfyUI模式: 升级GPU
