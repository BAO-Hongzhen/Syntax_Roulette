# 📦 项目文件说明

## 🎯 核心文件（必需）

- 快速预设

## 💻 代码文件

### text_to_image.py ⭐⭐⭐
**ComfyUI工作流封装类**

**功能：**
- TextToImageWorkflow类
- 工作流参数管理
- JSON配置生成

**用途：** 被其他文件导入使用

---

### gradio_text_to_image.py ⭐⭐⭐
**Gradio完整版本（备选）**

**功能：**
- 独立的Gradio应用
- 与main.py功能类似
- 可选择使用

**运行：**
```bash
python gradio_text_to_image.py
```

---

### streamlit_text_to_image.py ⭐⭐⭐
**Streamlit版本（备选）**

**功能：**
- 使用Streamlit框架
- 不同的交互方式

**运行：**
```bash
streamlit run streamlit_text_to_image.py
```

---

### text_to_image.json
**ComfyUI原始工作流配置**

**用途：** 参考配置，可在ComfyUI中导入

## 🎨 版本对比

### main.py vs gradio_text_to_image.py

| 特性 | main.py | gradio_text_to_image.py |
|------|---------|------------------------|
| 界面 | 简洁美观 | 功能完整 |
| 代码 | 独立完整 | 依赖text_to_image.py |
| 推荐 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 适合 | 所有用户 | 需要ComfyUI集成的用户 |

**建议：新用户使用 `main.py`**

---

### Gradio vs Streamlit

| 特性 | Gradio (main.py) | Streamlit |
|------|------------------|-----------|
| 易用性 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 美观度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 速度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| 分享 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| 推荐 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

**建议：优先使用Gradio版本（main.py）**
