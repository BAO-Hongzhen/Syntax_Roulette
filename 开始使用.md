# 🎯 开始使用 Syntax Roulette

## 方式1：最简单（推荐给新手）

### Windows用户
1. **双击** `启动应用.bat` 文件
2. 等待自动安装依赖（首次运行需要）
3. 浏览器会自动打开应用界面
4. 开始创作！

---

## 方式2：命令行启动

### 步骤1：安装依赖
打开终端（PowerShell或命令提示符），在项目目录下运行：

```bash
pip install gradio pillow numpy
```

### 步骤2：启动应用
```bash
python main.py
```

### 步骤3：打开浏览器
访问：`http://localhost:7860`

---

## 🔍 环境检测（可选）

如果不确定环境是否配置正确，运行：

```bash
python check_env.py
```

这会检查：
- ✅ Python版本是否符合要求
- ✅ 必需依赖是否已安装
- ✅ 可选依赖的安装状态

---

## 📖 使用教程

### 第一次使用

1. **输入描述文字**
   ```
   在左侧的"描述你想要的图像"框中输入，例如：
   一只可爱的橘猫坐在窗台上，阳光洒在它身上
   ```

2. **选择风格（可选）**
   ```
   在"艺术风格"下拉菜单中选择，如：
   - 默认
   - 写实
   - 动漫
   等
   ```

3. **调整参数（可选）**
   ```
   - 图像尺寸：512x512（推荐新手使用）
   - 质量等级：高
   ```

4. **点击生成**
   ```
   点击"🎨 生成图像"按钮
   等待几秒钟
   ```

5. **查看结果**
   ```
   生成的图像会显示在右侧
   可以点击"📥 下载图像"保存
   ```

## ⚙️ 参数说明

### 图像尺寸
- **256-384**: 快速预览，生成速度快
- **512-640**: 推荐使用，平衡质量和速度
- **768-1024**: 高质量，但生成较慢

### 质量等级
- **低**: 快速测试用
- **中**: 日常使用
- **高**: 推荐设置 ⭐
- **超高**: 最佳质量，细节最丰富

### 艺术风格
根据想要的效果选择：
- **默认**: 通用场景
- **写实**: 想要照片效果
- **动漫**: 想要卡通效果
- **油画/水彩**: 想要艺术感
- **赛博朋克**: 想要科幻感

## 🎨 创作技巧

### 技巧1：从简单开始
```
第1次：一只猫
第2次：一只橘色的猫
第3次：一只橘色的猫坐在窗台上
第4次：一只橘色的猫坐在窗台上，阳光洒在它身上
```
逐步添加细节，找到最佳效果！

### 技巧2：使用质量词
在描述后面添加：
```
, 高清, 细节丰富, 专业摄影, 最佳质量
```

### 技巧3：使用负面提示
在"不想要的元素"中输入：
```
模糊, 低质量, 变形, 噪点
```

### 技巧4：批量生成对比
1. 固定描述和参数
2. 勾选"使用随机种子"
3. 多次点击生成
4. 在历史记录中选择最好的

## 📚 历史记录

2. 点击"🔄 刷新历史"
3. 查看所有生成的图像
4. 可以点击查看每次的参数

## ❓ 遇到问题？

### 应用无法启动
```bash
# 检查环境
python check_env.py

# 重新安装依赖
pip install gradio pillow numpy --upgrade

# 再次尝试启动
python main.py
```

### 生成失败
- 检查描述是否为空
- 尝试更简单的描述
- 刷新浏览器页面
- 重启应用

### 图像质量不满意
- 提供更详细的描述
- 选择更高的质量等级
- 尝试不同的艺术风格
- 多次生成，选择最佳结果
